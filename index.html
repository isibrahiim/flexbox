flex-item properties

All properties that work with flex-item:


  • oreder
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self

order properties:


the order in which flex item appear in the flex container, by default all items have an order of zero, so order 1 is greater than the order of all other items and hence it appears in the very end

flex-grow properties:


the ability of flex item to grow if necessary
by default all flex item have flex grow value of zero
the item will not grow to fit the container

flex-shrink properties:


the ability of flex item to shrink if necessary
by default the value is 1
it implies that the item will shrink when necessary

flex-basis properties:


specifies the initial size of flex-item before the extra space in the container is distributed, this is used in a place of width property in flex layout
by default the flex-bases has a value of auto
it implies that the width of the item depends on their content
you can specify values in pixels, precentage or relative units

flex properties:


short hand for flex-grow, flex-shrink and flex-basis
flex-grow: 1;
flex-shrink: 0;
flelx-basis: 500px;
here is the code
flex: 1 0 500px;

align-self properties:


alignment of individual flex items
override of the align items value of flex container
default value is auto
align-item of container has default value of STRETCH
align-self of individual items has defaultvalue of AUTO;
