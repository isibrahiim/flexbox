flexbox

container

all properties that work with container:

  • display
  • flex-direction
  • flex-wrap
  • flex-flow
  • justfy-content
  • align-items
  • align-content
display

Display properties:

  • display:flex
  • display:inline-flex
flex-direction

flex-direction properties:

  • row ----default--
  • row-reverse
  • column
  • column-reverse
flex-wrap

flex-wrap properties:

  • nowrap-- default--
  • wrap -- in the container
  • wrap-reverse -- above row

    • vertical wrapping

    change the direction into columns and add a height

flex-flow

flex-flow properties

short hand for flex-direction and flex-wrap
by default flex-flow property is set to row and nowrap

justfy-content

justfy-content properties:

alignment along the main Axis
by default flex-start (to the left)

possible values:

  • flex-satart
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly
align-items

align-items properties:

alignment along the cross Axis

possible values

  • stretch is the default
  • flex-star
  • flex-end
  • center
  • baseline
align-content

align-content properties

aligns lines of content along the cross Axis
mixed of JUSTFY-CONTENT and ALIGN-ITEMS

possible values

  • stretch
  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
